## 简介

`ssc` 是万~~恶~~物之源，其内存储了全部 SteveScratchC 的接口函数供开发使用。

```lua
ssc.msgBox("A cute dialog :P")
```

## 变量

-|-|-

## 函数

函数名|用途|出现版本
-|-|-|
msgBox|弹出一个信息框|AFD v1

### msgBox(string msg)
#### 作用
弹出一个信息框
#### 用法
```lua
ssc.msgBox("A cute dialog :P")
```
#### 参数
msg：一个字符串，为信息框展示的内容。
#### 返回值
无
